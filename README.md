# Todoism

A simple todo app built with Typescript

### Prerequisites

Typescript

```
npm install -g typescript
```

### Installing

Install dependencies 

```
npm install
```

Complile and run

```
npm start
```

## Built With

* [Typescript](https://www.typescriptlang.org/)
* [Webpack](https://webpack.js.org/)
